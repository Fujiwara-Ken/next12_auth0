# Next12 と auth0 の starter example

Next.js(auth0)の連携サンプル

## ディレクトリ構成

すべてのファイルを `src` に格納。
components の中身は PJ 毎に自由に変えてください。

### 環境構築

## 1. `git clone`

```shell
git clone git@github.com:Fujiwara-Ken/next_example.git
```

## 2. `yarn install`

`yarn install`

```shell
# v1.22.15であることを確
yarn -v

#パッケージをインストール
yarn install
```

## 3. `yarn dev`

`yarn dev`でローカルサーバーを立ち上げ。

```shell
yarn dev
```
